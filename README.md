# 2e-rocket
2E Rocket - Extremely extra fancy new tab page

### Prerequisites
- It's a TypeScript project, do anything yourself to make it work ^^
### Usage
- Firstly, BUILD
- From the Chrome extension manager, turn on developer mode, and import the `build` folder.
- Play!
